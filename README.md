# PHP-MSF DEMO

PHP微服务框架即“Micro Service Framework For PHP”示例项目

# Notice

- php-msf-demo示例项目使用的是最新的php-msf代码，即dev-master分支，生产环境请使用release的相关版本。
- 在有些环境中执行`php server.php`，由于`inotify`的原因并不能自动热更新代码，请尝试使用`nodemon --exec "php" server.php`
